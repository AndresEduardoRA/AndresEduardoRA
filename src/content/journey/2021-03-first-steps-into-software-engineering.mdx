## Starting University

In March 2021, I'm beginning my Computer Engineering degree at Universidad Autónoma Gabriel René Moreno (UAGRM). I'm excited to learn programming and build software systems.

During the first weeks, I'm realizing that while university provides a solid foundation, the pace is slower than I expected. I want to learn faster and build real projects.

## Discovering Self-Directed Learning

I'm starting to explore resources beyond the classroom:

- MDN Web Docs (which is also motivating me to improve my English)
- YouTube tutorials from creators like MoureDev, Midudev, and Fazt
- Official documentation
- Google searches for specific problems

This is opening my eyes to how much knowledge is available online. I don't have to wait for classes to learn something new.

## Building My First Projects

I'm starting to build small projects to practice what I'm learning:

- Simple web pages with HTML, CSS, and JavaScript
- Basic CRUD applications
- Small APIs to understand backend concepts

Most of these early projects are learning exercises. I build something, learn from it, and then move on to the next challenge. I'm not keeping many of them because they're stepping stones, not portfolio pieces.

## Key Realizations

**Learning is a skill**: The most important thing I'm learning isn't a specific technology, but how to learn effectively on my own.

**Practice matters more than theory**: Reading about programming isn't enough. I need to write code every day.

**English is essential**: Most of the best resources are in English, which is pushing me to improve my language skills.

## Growth Focus

At this stage, I'm focusing on:

- Building a strong foundation in programming fundamentals
- Developing consistent study habits
- Learning how to debug and solve problems independently
- Understanding how web applications work end-to-end
